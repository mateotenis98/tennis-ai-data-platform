# Tennis AI Data Platform 🎾

> End-to-End AI & Data Engineering platform for extracting, processing, and predicting ATP tennis odds using Google Cloud Platform (GCP).

## 📌 Project Overview
This repository contains the pipeline to ingest live tennis data (Head-to-Head odds) from The-Odds-API, transform it, and eventually feed it into Machine Learning models for predictive analysis.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Cloud Provider:** Google Cloud Platform (GCP)
* **Storage:** Cloud Storage (Data Lake)
* **Data Source:** The-Odds-API
